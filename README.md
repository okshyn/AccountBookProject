# Family Account Book

Java Swing을 활용하여 개발한 가족 가계부 관리 프로그램입니다.

## 주요 기능

### 사용자
- 회원가입 및 로그인
- 수입/지출 입력
- 가계부 내역 조회
- 본인 예산 조회

### 관리자
- 가족 회원 관리
- 구성원별 예산 설정
- 가족 지출 현황 조회
- 연말정산 기능

## 개발 환경

- Java
- Java Swing
- Eclipse

## 실행 방법

1. 프로젝트 다운로드
2. LoginFrame.java 실행
