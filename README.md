# Family Account Book

Java Swing을 활용하여 개발한 가족 가계부 관리 프로그램입니다.

## 주요 기능

### 사용자
- 회원가입 및 로그인
- 수입/지출 입력
- 가계부 내역 조회
- 본인 예산 조회

### 관리자
- 가족 회원 관리
- 구성원별 예산 설정
- 가족 지출 현황 조회
- 연말정산 기능

## 테스트 계정

프로그램 기능 테스트를 위해 샘플 계정과 데이터를 미리 생성하였습니다.

## 테스트 데이터

프로그램 기능을 확인하기 위해 2025년 및 2026년의 샘플 가계부 데이터와 예산 데이터를 미리 입력하였습니다.

샘플 데이터는 다음 기능을 테스트하기 위해 사용됩니다.

- 가계부 입력 및 조회
- 월별 지출 현황 조회
- 구성원별 예산 설정
- 예산 대비 사용량 확인
- 연말정산 기능
